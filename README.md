# ChittyCommand

> Unified life management and action dashboard for the ChittyOS ecosystem.

ChittyCommand ingests data from 15+ financial, legal, and administrative sources — Mercury, Plaid, court dockets, mortgage servicers, utility portals — scores urgency with AI, and recommends or executes actions via APIs, email, or browser automation. It runs as a Cloudflare Worker with a React SPA frontend, Neon PostgreSQL for persistence, R2 for document storage, and a built-in MCP server exposing 43 tools to Claude.

**Domain**: `command.chitty.cc`
